# Lim-ID-Builder (Limbus Identity Builder)

『Limbus Company』のオリジナル人格（Identity）をブラウザ上で作成・編集・管理できる非公式のWebアプリケーションツールです。
このツールはブラウザ上で完結し、作成したデータはローカルに保存されるため、安全かつ手軽に二次創作人格のステータスやスキルを形にすることができます。

👉 **[アプリを開く（GitHub Pages）](https://getousugu.github.io/Lim-ID-Builder/)**

---

## 🎨 主な機能と使い方

### 1. キャラクターの作成と管理
トップページの「新規人格作成」ボタンから、新しいキャラクターデータを作成できます。
- **基本情報の登録**: 名前、所属囚人（イサン～グレゴール）、チーム名、シーズン、リリース日などを設定します。
- **画像のアップロード**: キャラクターの画像をアップロードしてアイコンとして設定できます（画像データはローカルにのみ保存されます）。
- **ステータスの設定**: 最大HP、速度の最小値・最大値、防御補正などを設定します。

### 2. 耐性と混乱区間の設定
- **属性耐性**: 斬撃、貫通、打撃の耐性をプルダウンで選択するか、カスタム倍率を直接入力できます。
- **混乱区間**: 敵からダメージを受けた際に混乱状態になる閾値を、HPの数値として複数設定できます。「＋追加」ボタンから任意の数を設定可能です。

### 3. スキルの詳細な構築
スキル1、スキル2、スキル3、そして守備スキルの構成を直感的にエディットできます。
- **基本設定**: スキルのベース威力、コインの追加威力、攻撃加重値、レベルボーナスを設定します。
- **罪悪属性と攻撃/守備タイプ**: 7つの罪悪属性と、斬撃/貫通/打撃/防御/回避/反撃などを選択。
- **コイン設定**: コインの枚数や、マッチ敗北時に破壊されない「破壊不能コイン」の設定が可能です。
- **スキルの説明とフレーバー**: 「使用時」「的中時」などのタグをショートカットで入力しながら、スキルの効果説明やフレーバーテキストを記述できます。

### 4. パッシブとパニック、精神力の条件設定
- **バトル / サポートパッシブ**: 共鳴や保有など、発動条件と罪悪属性を設定したパッシブを作成できます。
- **パニックと精神力条件**: パニック時の特殊な行動や、精神力が増減する独自の条件を記述できます。

### 5. 用語辞書（サジェスト）機能
「出血」や「火傷」のような既存の状態異常や、オリジナルのバフ/デバフ効果を辞書として登録できます。
- スキル説明やパッシブのテキストエリアで `@` と入力すると、登録された用語がサジェストされます。
- サジェストから選択すると `[[用語名]]` の形式で入力され、プレビュー画面では自動的にアイコン付きのハイライト表示（マウスオーバーで詳細確認）に変換されます。

### 6. リッチなプレビューとエクスポート
- **プレビューモード**: 編集中の人格を、ゲーム内に近いデザインのカードUIで確認できます。完成した人格のスクリーンショットを撮って共有するのに最適です。
- **JSON出力と読み込み**: 作成したキャラクターは「JSONエクスポート」からローカルにファイルとして保存できます。トップページの「JSONインポート」から読み込むことで、他のブラウザや端末にデータを移行したり、他人が作成した人格データを読み込むことが可能です。

## 🛠 用語辞書のカスタマイズについて

トップページの「用語辞書管理」から、アプリ全体で共有される用語リストを編集できます。
- **プラス（バフ） / マイナス（デバフ） / ニュートラル** の種別を設定すると、プレビュー時のテキストカラーが自動で割り当てられます。
- 独自のアイコン設定や、プレビュー時に表示される詳細な説明文、フレーバーテキストを登録できます。
- 他ユーザーのJSONをインポートした際、同じ名前の用語でも定義が異なる場合は、競合解決のポップアップが表示され、「ローカルを優先」「上書き」「別名として両方残す」から選択できます。

## ⚠️ 動作環境と注意事項

- **推奨環境**: Google Chrome, Mozilla Firefox, Apple Safari などのモダンブラウザ。
- **データ保存場所**: アプリケーションのデータ（キャラクター情報、用語辞書、画像）はすべてご利用のブラウザの「ローカルストレージ」に保存されます。ブラウザのキャッシュや履歴を削除するとデータが消失する可能性があるため、完成したデータは定期的に「JSONエクスポート」で保存することをお勧めします。
- **免責事項**: 本ツールはファンによる非公式の二次創作ツールであり、Project Moon様および公式ゲーム『Limbus Company』とは一切関係ありません。

### 開発者向け (ローカル環境での実行)

リポジトリをクローンして独自のカスタマイズを行う場合は以下の手順で起動してください。

```bash
# 依存関係のインストール
npm install

# 開発用サーバーの起動 (localhost:3000)
npm run dev

# プロダクションビルド
npm run build
```

# Lim-ID-Builder (Limbus Identity Builder)

『Limbus Company』のオリジナル人格（Identity）を、ブラウザ上で作成・編集・管理できる非公式Webアプリケーションです。

すべての処理はブラウザ内で完結し、作成したデータはローカルに保存されます。インストール不要・アカウント登録不要で、安全かつ手軽に二次創作人格のステータスやスキルを形にできます。

> 本ツールのコーディングはAIによって行われています。不具合を見つけた場合は [Issues](../../issues) からご報告ください。対応にはお時間をいただく場合がありますが、可能な限り対応いたします。
> UIは今後も改善を続けていく予定です。現時点では発展途上の部分も多いですが、ご理解のうえご利用ください。

👉 **[アプリを開く（GitHub Pages）](https://getousugu.github.io/Lim-ID-Builder/)**

## 主な機能

### 1. キャラクターの作成と管理

トップページの「新規人格作成」ボタンから、新しいキャラクターデータを作成できます。

- **基本情報の登録**：名前、所属囚人（イサン～グレゴール）、チーム名、シーズン、リリース日などを設定
- **画像のアップロード**：キャラクターアイコンとして画像を設定可能（画像データはローカルにのみ保存されます）
- **ステータスの設定**：最大HP、速度の最小値・最大値、防御補正などを設定

### 2. 耐性と混乱区間の設定

- **属性耐性**：斬撃・貫通・打撃の耐性をプルダウンで選択、またはカスタム倍率を直接入力
- **混乱区間**：敵からダメージを受けた際に混乱状態になるHP閾値を、「＋追加」ボタンから任意の数だけ設定可能

### 3. スキルの詳細な構築

スキル1・スキル2・スキル3、および守備スキルの構成を直感的に編集できます。

- **基本設定**：ベース威力、コインの追加威力、攻撃加重値、レベルボーナス
- **罪悪属性と攻撃／守備タイプ**：7つの罪悪属性、および斬撃／貫通／打撃／防御／回避／反撃などを選択
- **コイン設定**：コインの枚数、マッチ敗北時に破壊されない「破壊不能コイン」の設定
- **スキルの説明とフレーバー**：「使用時」「的中時」などのタグをショートカットで入力しながら、効果説明・フレーバーテキストを記述

### 4. パッシブ・パニック・精神力条件の設定

- **バトル／サポートパッシブ**：共鳴や保有など、発動条件と罪悪属性を設定したパッシブを作成
- **パニックと精神力条件**：パニック時の特殊行動や、精神力が増減する独自条件を記述

### 5. 用語辞書（サジェスト）機能

「出血」「火傷」のような既存の状態異常や、オリジナルのバフ／デバフ効果を辞書として登録できます。

- スキル説明やパッシブのテキストエリアで `@` と入力すると、登録済みの用語がサジェスト表示されます
- サジェストから選択すると `[[用語名]]` の形式で入力され、プレビュー画面ではアイコン付きハイライト表示に自動変換されます（マウスオーバーで詳細確認可）

### 6. リッチなプレビューとエクスポート

- **プレビューモード**：編集中の人格を、ゲーム内デザインに近いカードUIで確認可能。完成した人格のスクリーンショットを撮って共有するのに最適
- **JSON出力／読み込み**：「JSONエクスポート」で作成データをファイル保存。「JSONインポート」で他ブラウザ・他端末へのデータ移行や、他ユーザーが作成した人格データの読み込みが可能

## 用語辞書のカスタマイズ

トップページの「用語辞書管理」から、アプリ全体で共有される用語リストを編集できます。

- **プラス（バフ）／マイナス（デバフ）／ニュートラル** の種別を設定すると、プレビュー時のテキストカラーが自動で割り当てられます
- 独自のアイコン、プレビュー時に表示される詳細説明、フレーバーテキストを登録可能
- 他ユーザーのJSONをインポートした際、同名で定義が異なる用語がある場合は競合解決のポップアップが表示され、「ローカルを優先」「上書き」「別名として両方残す」から選択できます

## 動作環境と注意事項

- **推奨ブラウザ**：Google Chrome / Mozilla Firefox / Apple Safari などのモダンブラウザ
- **データ保存場所**：キャラクター情報・用語辞書・画像はすべてブラウザの「ローカルストレージ」に保存されます

⚠️ ブラウザのキャッシュや履歴を削除するとデータが消失する可能性があります。完成したデータはこまめに「JSONエクスポート」でバックアップすることを推奨します。

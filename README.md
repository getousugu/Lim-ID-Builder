# Lim-ID-Builder (Limbus Identity Builder)

Limbus Companyのオリジナル人格（Identity）を作成・編集・管理するためのWebアプリケーションツールです。

## 主な機能

- **キャラクター管理**: オリジナルの人格（Identity）を作成・保存・管理します。
- **ステータス・耐性設定**: HP、速度、防御補正、属性耐性、混乱区間などを細かく設定できます。
- **スキル・パッシブ構築**: スキル1～3、守備スキル、各種パッシブスキルを直感的に作成可能。コインの効果や特殊タグにも対応。
- **用語辞書・サジェスト機能**: 状態異常などの用語を辞書で管理。スキル説明入力時に `@` を入力することで用語サジェストが利用でき、プレビュー時には自動で色付きのツールチップ表示に変換されます。
- **プレビュー機能**: 作成した人格の情報を、ゲーム内に近いリッチなUIで確認できます。
- **インポート / エクスポート**: 作成したデータをJSON形式でファイル出力し、他の環境にインポートすることができます。用語辞書の競合解決にも対応しています。

## 開発・セットアップ

本プロジェクトは [Vite](https://vitejs.dev/) を使用して構築されています。

### 依存関係のインストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:3000/` などにアクセスして動作を確認できます。

### プロダクションビルド

```bash
npm run build
```

## 使用技術

- HTML / Vanilla CSS / Vanilla JavaScript
- Vite (Dev server & Bundler)

## 注意事項

- 本ツールはファンによる非公式の二次創作ツールであり、Project Moon様および公式ゲーム『Limbus Company』とは一切関係ありません。
- アプリケーションのデータはローカルストレージに保存されます。重要なデータはエクスポートしてバックアップしてください。

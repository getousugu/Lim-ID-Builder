# Limbus キャラクタービルダー 仕様書 v2

---

## 概要

Limbus Company のファン向けキャラクタービルダーWebアプリ。
オリジナルキャラクター（人格）を自由に設計し、Dante's Plannerライクなレイアウトで表示できる。
データはブラウザにローカル保存され、JSONファイルとしてエクスポート／インポートできる。

---

## 画面構成

```
┌──────────────────────────────────────────────┐
│ ヘッダー（アプリ名）                [人格一覧]    │
├───────────────────┬──────────────────────────┤
│ 左パネル          │ 右パネル                  │
│ - キャラ画像      │ [スキル1][スキル2][スキル3][守備] │
│ - 基本情報        │                          │
│ - ステータス      │ 選択中タブのスキルカード群    │
│ - 耐性情報        │ （複数スキルは縦に連続表示） │
│ - 混乱情報        │                          │
│ - 特性キーワード  │ パッシブ一覧               │
│ - シーズン/日付   │                          │
├───────────────────┴──────────────────────────┤
│ [編集⇔プレビュー切替] [JSON保存/読込]           │
└──────────────────────────────────────────────┘
```

「人格一覧」ボタンを押すと、編集画面から離れて15章の一覧画面に遷移する。

## 6. スキル（スキル1〜3 ＋ 守備）

各タブ（スキル1〜3／守備）はスキルの配列を持ち、複数のスキルを登録できる（例：スキル3-1, 3-2, 3-3…）。
タブ内では登録順に全スキルカードが縦に連続表示される（並べ替え可能）。

### 6-1. スキル共通情報

| フィールド | 型 | 備考 |
|---|---|---|
| スキル名 | テキスト | |
| スキルレベルボーナス | 数値（+N） | |
| 罪悪属性 | 選択（単一） | 嫉妬・憂鬱・憤怒・色欲・怠惰・暴食・傲慢 の7種から1つ |
| 攻撃属性 | 選択 | 斬撃／貫通／打撃。**通常攻撃スキル・反撃・マッチ可能反撃のみ入力可**。防御／回避／マッチ可能ガードでは入力欄非表示 |
| コイン数 | 数値（1〜10） | |
| コイン威力 | 数値（自由入力、負数可） | 負数入力で実質的な「減算コイン」を表現する（種類としては分けない） |
| コインの種類（コインごとに設定） | 選択 | 通常(金色) ／ 破壊不能(赤色)。破壊不能は**末尾から連続**でのみ指定可能（例：末尾1枚のみ、末尾2枚、全数、いずれもOK。間に挟む・先頭側のみは不可）。UI上は「破壊不能コインの枚数」を1つの数値入力にし、末尾から自動着色する |
| ベース威力 | 数値 | |
| 攻撃レベル | 数値（自由入力） | マッチ威力判定に使う基準値。敵との差5につき最終威力±1 |
| 攻撃加重値 | 数値（1〜10、プルダウン） | 初期値1。対象にできるスロット数。UI上は■の数で表示 |
| 守備スキルタイプ（守備欄の場合のみ） | 選択 | 防御／反撃／回避／マッチ可能ガード／マッチ可能反撃 |
| 特殊タグ | チェックボックス（複数可）＋自由入力欄 | マッチ不可／敵味方識別不可／対象変更不可。チェック時、効果テキスト最上位に赤文字で表示。それ以外のタグは自由入力欄に追記可能 |
| フレーバーテキスト | テキスト（複数行、任意） | 世界観・設定の説明文。空欄可 |

### 6-2. スキル効果テキスト

| フィールド | 型 | 備考 |
|---|---|---|
| 効果テキスト | リッチテキストエリア（複数行・箇条書き対応） | 1スキルにつき1つの自由記述欄。用語辞書（8章）のインライン参照可 |
| 条件タグ挿入 | ショートカットボタン | 使用前／使用時／攻撃後／的中時／撃破後／回避後／クリティカル表面的中時／再使用表面的中時／戦闘開始時／その他（自由入力）をクリックでカーソル位置に`[○○]`を挿入するだけ。データ構造上は本文の一部 |

### 6-3. コイン効果

コインI・II・III・IV…それぞれに効果ブロックを追加可能（最大10枚）。

| フィールド | 型 | 備考 |
|---|---|---|
| コイン番号 | I / II / III / IV… | ローマ数字の丸バッジで表示 |
| コイン種類 | 選択 | 通常(金色) ／ 破壊不能(赤色)。6-1の破壊不能枚数指定と連動 |
| 効果テキスト | リッチテキストエリア（複数行対応、条件なしOK） | 条件タグ挿入ショートカット（的中時／表面的中時／クリティカル的中時／再使用表面的中時／その他）も文中挿入方式 |

**破壊不能コインの自動表示（システム挙動）**
コイン種類が「破壊不能」の場合、そのコインの効果テキストエリアの**先頭に「🔴破壊不能コイン」という行をシステムが自動挿入・固定表示**する（編集不可、ユーザーの手入力は不要）。上部のコイン枚数アイコン列でも該当コインを赤色で表示する。

## 7. パッシブ

属性は2種以上の複数選択可（例：暴食×1 怠惰×1 憂鬱×1 保有）。

### 7-1. 通常パッシブ

| フィールド | 型 | 備考 |
|---|---|---|
| パッシブ名 | テキスト | |
| 共鳴数もしくは保有資源数 | 数値（任意） | 空欄可（複数種の場合は種類ごとに数値を持てる） |
| 共鳴もしくは保有資源属性 | 選択（複数可） | 罪悪属性7種から選択 |
| 効果テキスト | テキスト（複数行） | 共鳴・保有等の条件を一切不要にすることも可能 |
| フレーバーテキスト | テキスト（複数行、任意） | |

パッシブは複数追加可能。

**共鳴バッジの自動表示**：「共鳴数もしくは保有資源数」と「共鳴もしくは保有資源属性」の両方が入力されている場合のみ、パッシブ名の下にアイコン付きバッジ（例：🔥×3 共鳴）を自動生成する。未入力ならバッジ非表示。ユーザーが手動でテキストに書く必要はない。

### 7-2. サポートパッシブ

7-1と同一構造（パッシブ名／共鳴数or保有資源数／共鳴or保有資源属性／効果テキスト／フレーバーテキスト）。共鳴バッジの自動表示ルールも同様。

## 8. 用語辞書管理

アプリ全体で共有する用語データベース。バフ・デバフだけでなく、状態異常・固有リソース・固有アイテムなど、スキルテキストからインライン参照したい用語全般を管理する。JSONで管理、キャラJSONへの同梱、ローカル保存に対応。

### 用語1件あたりの構造

| フィールド | 型 | 備考 |
|---|---|---|
| ID | 自動 | 内部管理用 |
| 名前 | テキスト | 例：「呼吸」「出血」「ムクエ房-イェ[鋭]二式抜刀」 |
| 種別 | 選択 | バフ／デバフ／状態異常／固有リソース／固有アイテム／特殊／その他 |
| カラー | カラーピッカー | テキスト表示時の色 |
| 効果説明テキスト | テキスト（複数行） | ホバー表示用 |
| アイコン画像 | 画像アップロード | 任意 |

### スキルテキスト内への挿入方法

- 編集中に `@` から入力すると候補サジェスト
- 選択するとカラー付きのインラインリンクとして挿入される
- プレビューでホバーすると効果説明がポップアップ表示

## 10. 精神力（基本精神力増減条件）

増加条件・減少条件それぞれ、条件テキストを複数行で自由入力。初期値として以下を入力しておく。

**基本精神力増加条件**
- マッチ勝利時、マッチ回数に比例して増加（増加量：基本値10、2マッチ目から1マッチにつき20%ずつ増加）
- 敵討伐時、倒した敵のレベルが自分のレベル以上の場合、10増加
- 味方が倒した敵のレベルが自分のレベル以上の場合、5増加

**基本精神力減少条件**
- 味方死亡時、死亡した味方のレベルが自分のレベル以上の場合、レベル差によって減少（減少量：基本値10、レベル差につき10ずつ増加）

| フィールド | 型 |
|---|---|
| 増加条件テキスト | 複数行テキスト |
| 減少条件テキスト | 複数行テキスト |

## 11. データの保存・読み込み

### 11-1. ローカル保存（自動）

| 対象 | 保存先 | 備考 |
|---|---|---|
| 作成中のキャラクター一覧 | ブラウザのローカルストレージ | 編集の都度自動保存。複数キャラを並行して作業できる |
| 用語辞書（8章） | ブラウザのローカルストレージ | キャラクターと独立して永続化。他のキャラクター作成時にも再利用できる |

### 11-2. JSONエクスポート/インポート（手動）

| 機能 | 方式 | 備考 |
|---|---|---|
| エクスポート | JSONファイルダウンロード | 1キャラ＝1ファイル。画像はBase64で埋め込み。参照している用語辞書の項目も全てそのキャラJSONに同梱し自己完結させる |
| インポート | JSONファイルアップロード | 読み込んだキャラをローカル保存の一覧に追加 |
| 用語の重複解決 | 確認ダイアログ | インポートしたJSON内の用語名がローカル辞書に既存かつ内容が異なる場合、都度ダイアログを表示。「ローカル辞書を使う」「インポート内容で上書きする」「別名として両方残す」から選択 |

## 12. スキルバッジ（アイコン）仕様

キャラクター固有イラストを用意せず、以下のルールでスキルバッジを自動生成する。

### 12-1. バッジ構成

| 要素 | 内容 |
|---|---|
| 形状 | 八角形（オクタゴン） |
| 背景色 | 罪悪属性の色（下記カラーテーブル） |
| アイコン | 攻撃属性（斬撃／貫通／打撃）または守備タイプ（防御／回避／マッチ可能ガード）のSVGシルエット。色分けなし、単色（金系）で統一 |

### 12-2. 罪悪属性カラーテーブル

| 罪悪属性 | 色 |
|---|---|
| 憤怒 | 赤 |
| 色欲 | 橙 |
| 怠惰 | 黄 |
| 暴食 | 緑 |
| 憂鬱 | 水色 |
| 傲慢 | 青 |
| 嫉妬 | 紫 |

罪悪属性は単一選択のため、バッジ色は必ず1色に決まる。

### 12-3. アイコンアセット

| 守備スキルタイプ | アイコン | 状態 |
|---|---|---|
| 通常攻撃（斬撃／貫通／打撃） | 各属性の専用シルエット | ユーザー提供済み |
| 防御 | 盾（専用アイコン） | 提供待ち |
| 回避 | 一回転する矢印（専用アイコン） | 提供待ち |
| マッチ可能ガード | 盾＋プラスマーク（専用アイコン） | 提供待ち |
| 反撃 | 攻撃属性アイコンを流用 | 専用アイコンなし（対応済み） |
| マッチ可能反撃 | 攻撃属性アイコンを流用 | 専用アイコンなし（対応済み） |

## 13. スキルカードのビジュアルレイアウト仕様

```
┌───────────────────────────────────────┐
│ [+N バッジ]              [コイン●●]     │
│  N   [八角形バッジ]  ┌─────────────┐    │
│ (威力) (攻撃属性SVG)  │ スキル名バナー │    │
│                       └─────────────┘    │
│  [⚔/🛡] 攻撃レベル 攻撃加重値 ■         │
│                                         │
│  効果テキスト（条件なし部分が最初）        │
│  [使用時] 効果テキスト                    │
│                                         │
│  I  [的中時] コイン効果テキスト            │
│  II [的中時] コイン効果テキスト            │
└───────────────────────────────────────┘
```

### 13-1. 要素と対応データ

| UI要素 | 対応するデータ | 備考 |
|---|---|---|
| +Nバッジ | スキルレベルボーナス（6-1） | 八角形バッジに重なる位置 |
| 大きい数字 | ベース威力（6-1） | |
| 八角形バッジ（背景色＋アイコン） | 罪悪属性（背景色）＋攻撃属性/守備タイプ（アイコン） | 12章の仕様通り |
| コインアイコン列 | コイン数＋コイン種類（6-1） | 通常＝金丸、破壊不能＝赤丸 |
| スキル名バナー | スキル名（6-1） | 背景色は罪悪属性に連動 |
| ⚔/🛡＋数字 | 攻撃レベル（6-1） | アイコンは攻撃/守備の固定システムアイコン（属性色なし） |
| 攻撃加重値 ■ | 攻撃加重値（6-1） | ■の数＝加重値の数値 |
| 効果テキスト全体 | 効果テキスト（6-2） | 自由記述、条件タグは文中の`[○○]`表記として表示 |
| I/II等の丸数字ブロック | コイン効果（6-3） | コイン番号ごとにグルーピング表示 |

### 13-2. 複数スキルの表示

タブ内に複数スキルが登録されている場合、上記カードを登録順に縦に連続して表示する。

## 14. 人格一覧画面

ヘッダーの「人格一覧」ボタンから遷移する画面。11-1のローカル保存に蓄積されている全キャラクター（完成・作業途中問わず）を一覧表示する。

### 14-1. 一覧に表示する情報

| 項目 | 内容 |
|---|---|
| サムネイル | キャラクター画像（未設定ならプレースホルダー） |
| キャラクター名 | 未入力なら「（名称未設定）」 |
| 所属囚人 | |
| 所属・チーム名 | |
| 最終更新日時 | |

### 14-2. カードごとの操作

| 操作 | 内容 |
|---|---|
| 開く | 編集画面に遷移し続きを編集 |
| 複製 | 同内容のコピーをローカル保存に新規追加 |
| 削除 | 確認ダイアログの上でローカル保存から削除 |
| JSONエクスポート | そのキャラクター単体を即座にJSON出力 |

### 14-3. 絞り込み・並び順

| 機能 | 内容 |
|---|---|
| 絞り込み | 所属囚人（12人格）でフィルタ。「すべて」＋12人格で切り替え |
| 並び順 | 最終更新日時順（新しい順）をデフォルト |
| 新規作成 | 一覧画面に「新規作成」ボタンを設置し、空のキャラクターで編集画面に遷移 |
